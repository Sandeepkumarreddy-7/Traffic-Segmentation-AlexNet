# TrafficVision

## Overview

TrafficVision is a semantic segmentation project that uses a Modified AlexNet-inspired encoder with a lightweight decoder and skip connections to segment traffic scenes from the BDD100K dataset.

## Goal

To perform pixel-wise semantic segmentation of traffic scenes while preserving spatial information using an improved AlexNet architecture.

## Technologies

- Python
- PyTorch
- OpenCV
- Albumentations
- TensorBoard
- Gradio
